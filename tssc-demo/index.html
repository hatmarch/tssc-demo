Secure Software Factory Demo (Dev Mode)

Overview

This is a demo to show Red Hat’s interpretation of the Department of Defense’s Software Enterprise DevSecOps Reference Design

It makes use of the following components:

  1. The Ploigos Operator: Red Hat sponsored operator for creating a trusted software supply chain

  2. Rekor: An immutable signature transparency log to provide further provenance and auditability for resources created by Ploigos
